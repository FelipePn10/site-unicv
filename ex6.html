Perguntas Frequentes

O que é HTML?

Linguagem que define a estrutura de uma página web por meio de tags.

O que é CSS?

Linguagem responsável pelo estilo visual: cores, fontes e layouts.

Preciso saber JavaScript para criar sites?

Não obrigatoriamente. Com HTML e CSS já dá pra criar páginas bem completas.

O que é responsividade?

Capacidade do site de se adaptar a diferentes tamanhos de tela.
